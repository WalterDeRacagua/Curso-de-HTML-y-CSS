2. TIPOS DE HOSTING
-------------------------------------------

Objetivo: hacer una clasificación de los hostings a utilizar.

Dependiendo de la necesidad y de los requerimientos de cada sitio web debemos saber qué hosting es el mejor.

	1. HOSTING COMPARTIDO: Un servidor con un montón de webs. Comparten recursos. El que más usuarios atrae, el que más afluencia tiene, más recursos consume. Este tipo de Hosting tiene un panel de control para gestionar nuestra web. Es gestionado por el proveedor. Es para empresas más pequeñitas que no tengan tanta afluencia, tantas visitas por así decirlo.

	2. VPS Virtual Private Server o Servidor Privado Virtual. Funciona distinto a HOSTING PRIVADO pero son parecidos. Separa el servidor en varias partes, no lo mete entero en un sitio. Cada parte es distinta, es como si fueran servidores distintos. Dentro de cada servidor puede haber varias zonas virtuales. El rendimiento de cada uno NO se ve afectado por otros. El rendimiento de una página aquí NO DEPENDE DE OTRAS PÁGINAS, en Hosting compartido SI. Es para empresas medianas con mucho tráfico de afluencia.
	
	3. HOSTING DEDICADO: Ofrece TOOOODO un servidor PARA UN SOLO CLIENTE. Ideal para páginas webs que manejan muchos datos. Para empresas muy importantes que generan mucha afluencia en su página. 

	4. CLOSE HOSTING: Distribuye tu web en un montón de servidores lo que hace que todo sea mucho más rápido, que sea mucho más escalable y flexible. Por ejemplo, para una página que vende productos que, en un Black Friday va a explotar de clientes. 

	5. HOSTING ESPECIALIZADO: Hosting optimizado para cosas específicas. Hay varios tipos de Hosting Especializados para distintos servicios.
